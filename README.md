# 文件编码检测与转换工具

一个基于 Python + tkinter 开发的图形化文件编码检测与转换工具。

## 功能特性

- **文件选择**：支持选择单个文件或遍历整个文件夹
- **编码检测**：自动识别文件当前的编码类型，显示置信度
- **编码转换**：支持转换为多种常见编码格式
- **批量处理**：支持批量检测和转换
- **安全选项**：可选择备份原文件，避免数据丢失

## 支持的目标编码

- UTF-8 / UTF-8-SIG（带BOM）
- GBK / GB2312 / GB18030（中文编码）
- BIG5（繁体中文）
- UTF-16 / UTF-32（各种字节序）
- ISO-8859-1 / WINDOWS-1252
- SHIFT_JIS / EUC-JP / EUC-KR

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 运行程序

```bash
python main.py
```

### 操作步骤

1. **添加文件**
   - 点击"选择文件"按钮选择单个或多个文件
   - 或点击"选择文件夹"按钮遍历整个文件夹

2. **检测编码**
   - 选中要检测的文件（可多选）
   - 点击"检测编码"按钮或按 F5

3. **转换编码**
   - 选中要转换的文件
   - 在"目标编码"下拉框中选择目标编码
   - 设置转换选项（备份、覆盖等）
   - 点击"转换编码"按钮或按 F6

4. **批量转换**
   - 直接点击"批量转换"按钮转换列表中的所有文件

### 视觉特性
- **卡片式布局**：白色卡片在浅灰背景上，层次分明
- **圆角按钮**：6px 圆角，带悬停效果和轻微阴影
- **分隔线**：细线分隔不同功能区域，清晰明了
- **简洁列表**：去除斑马纹，采用悬停高亮效果
- **轻量状态栏**：底部简洁状态栏，显示实时状态
- **彩色日志**：不同级别日志显示不同颜色（蓝/绿/橙/红）

### macOS 风格控件
- **MacOSButton**：圆角矩形按钮，带悬停动画和阴影效果
- **MacOSSwitch**：切换开关（绿色表示开启）
- **MacOSSegmentedButton**：分段控制器
- **卡片容器**：白色背景，清晰的功能分区

### 界面缩放功能
点击工具栏 **⚙️ 设置** 按钮，可打开界面设置对话框：

- **字体大小调节**：范围 10-20pt，默认 14pt
- **界面缩放调节**：范围 80%-150%，默认 100%
- **实时预览**：调整时界面实时变化
- **自动保存**：设置自动保存到 `ui_config.json`
- **重置默认**：一键恢复默认设置

默认界面尺寸：
- 默认窗口：1400×900（可缩放）
- 按钮尺寸：110×28 像素（紧凑设计）
- 字体大小：14pt
- 列表行高：36 像素

## 更新日志

### 2025-03-17
- **修复**: 中文编码检测错误导致转换后中文变乱码的问题
  - 问题: `chardet` 库有时将 GBK/GB18030 编码的中文文件误判为 ISO-8859-1 或 WINDOWS-1252
  - 解决方案: 添加中文编码优先处理机制，当检测置信度低或检测到西欧编码时，优先尝试 GB18030/GBK/GB2312/BIG5 编码
  - 添加 `_is_valid_chinese_text()` 方法验证解码结果，避免中文变成乱码
